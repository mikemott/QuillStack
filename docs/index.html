Discover QuillStack

Take a guided tour through the features that make capturing and organizing handwritten notes effortless.

Chapter I

Capture with Precision

Point your camera at any handwritten note. QuillStack's intelligent capture system automatically detects and enhances your handwriting for optimal OCR accuracy.

  • Overlay templates guide your writing
  • Import from your photo library
  • Real-time preview and crop
  • Multi-page document support
⚡
Chapter II

OCR with Confidence

Our three-pronged OCR system combines Apple's latest Vision framework, intelligent spell correction, and word-level confidence tracking to deliver exceptional accuracy.

  • Per-word confidence scores
  • Low-confidence words highlighted
  • Tap to see alternatives
  • Learns from your corrections

  • Each type routes to a purpose-built view

Chapter IV

AI Enhancement

Tap "Enhance" to have Claude clean up your OCR text, fix remaining errors, and improve formatting—all while preserving your original intent.

  • Powered by Claude API
  • Context-aware corrections
  • Works offline (queues for later)
  • Your API key, your control

Chapter V

Handwriting to GitHub Issues

Write a feature idea with #feature#, and QuillStack transforms your rough notes into a well-structured GitHub Issue with one tap.

  • LLM refines your prompt
  • Suggests labels automatically
  • Pick any repository
  • Perfect for Claude Code workflows
